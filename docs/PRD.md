# 깜짝링크공유앱(TrickPal) PRD

## 1. 프로젝트 개요

TrickPal은 장난치기 좋아하는 10-20대 사용자들을 위한 깜짝 링크 공유 앱입니다. 사용자는 상대방을 속이거나 호기심을 자극하는 메타이미지를 선택하고, 해당 링크를 공유하여 재미있는 영상이나 사운드로 깜짝 놀라게 할 수 있습니다. 최대한 단순한 UI를 제공하여 누구나 쉽게 사용할 수 있도록 설계되었습니다.

## 2. 유저 플로우

1. **메인 페이지 접속**: 사용자가 앱에 접속하면 간단한 인터페이스의 메인 페이지가 나타납니다.
2. **메타이미지 선택**: 제공된 카테고리에서 상대방을 속이거나 호기심을 자극할 수 있는 메타이미지를 선택합니다.
3. **깜짝 콘텐츠 선택**: 메타이미지와 매칭될 깜짝 영상이나 사운드를 선택합니다.
4. **링크 생성 및 공유**: 선택한 콘텐츠로 커스텀된 링크를 생성하고, 이를 복사하거나 바로 공유합니다.
5. **수신자 경험**: 링크를 받은 수신자는 메타이미지를 보고 호기심에 링크를 클릭하며, 깜짝 콘텐츠를 경험하게 됩니다.

## 3. 핵심 기능

### 3.1 메타이미지 선택 기능

카테고리 제공: 긴급/중요 메시지, FOMO 유발, 호기심 자극, 미신/행운 등 다양한 카테고리의 메타이미지를 제공합니다.

#### 예시 메타이미지:

- **긴급/중요 메시지**:
  - "회사에서 급한 연락입니다"
  - "[읽지 않은 메시지] 엄마: 지금 당장 확인해"
- **FOMO 유발**:
  - "당신만을 위한 특별 이벤트 초대장"
  - "친구가 당신의 TMI를 공유했습니다"
- **호기심 자극**:
  - "당신의 인스타그램 계정이 해킹되었을 수 있습니다"
  - "당신을 몰래 좋아하는 사람의 메시지"
  - "당신의 전생 결과가 도착했습니다"
- **미신/행운**:
  - "행운의 편지: 1분 안에 공유하지 않으면..."
  - "오늘의 운세: 클릭해서 확인하세요"

### 3.2 깜짝 콘텐츠 선택 및 재생 기능

영상 및 사운드 라이브러리: 미리 준비된 다양한 깜짝 영상과 사운드를 제공합니다.

#### 예시 콘텐츠:

- **점프스케어 콘텐츠**:
  - 갑자기 좀비가 화면을 덮치는 영상
  - 귀신이 천천히 다가오다가 급격히 달려드는 영상
  - 평화로운 영상이 갑자기 무서운 장면으로 전환
- **유머러스 콘텐츠**:
  - "너 지금 잠?" 메시지가 뜨다가 갑자기 크게 "야!!!"
  - 귀여운 동물 영상이 갑자기 웃긴 밈으로 전환
  - 잔잔한 음악이 갑자기 트로트/클럽음악으로 변경
- **트렌드 활용**:
  - 유행하는 틱톡 챌린지 패러디
  - 인기 밈/드라마 장면 활용한 깜짝 영상

### 3.3 링크 생성 및 공유 기능

- **커스텀 링크 생성**: 선택한 메타이미지와 깜짝 콘텐츠로 구성된 링크를 생성합니다.
- **간편 공유**: SNS, 메신저 등을 통해 쉽게 공유할 수 있도록 공유 기능을 제공합니다.

### 3.4 사용자 경험 최적화

- **단순한 UI**: 복잡한 요소를 배제하고 직관적인 인터페이스를 제공합니다.
- **반응형 디자인**: 다양한 기기에서 최적의 사용자 경험을 제공합니다.
